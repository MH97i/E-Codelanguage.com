Getting Started with E-CODE

Welcome to E-CODE! Here’s how you can get started with using the E-CODE programming language:

  1. Install E-CODE on your system by following the Download page.
  2. Set up your first E-CODE project by creating a new file with the .ec extension.
  3. Write your first line of code!
  4. Refer to the Syntax page for more information.
