E-CODE Syntax

Here’s an overview of the basic syntax for E-CODE:

  • Statements: Every line of code in E-CODE is a statement. Example: 
    Print "Hello World";
  • Variables: You can declare variables like this: 
    Initialize Variable "score" as Integer;
  • Functions: Functions are declared with the keyword "Function": 
    Function CalculateScore;
  • Conditions: Use "If" and "When" for conditional statements: 
    If score > 10 then Print "Great";
  • Loops: Use "For" for loops: 
    For i = 1 to 10 then Print i;
